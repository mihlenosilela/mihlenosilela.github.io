# Reflection: Using GitHub Pages

## Situation
I was required to deploy my digital portfolio online using GitHub Pages so that
it could be accessed by lecturers and potential employers.

## Task
My task was to correctly configure GitHub Pages, ensure all Markdown files were
rendered properly, and verify that my portfolio was publicly accessible this was also going to help me get marks for my PRP module.

## Action
I created a repository named after my GitHub username, enabled GitHub Pages
from the repository settings I encoded my CV actually the entire Portfolio using Markdown language, and tested the live site thoroughly. I checked
navigation, links, and video embedding to ensure everything worked correctly but i was having difficulties with adding my mock-interview video because it was a big file so it did not want to show on the portfolio instead it added a link that redirects you to it.

## Result
My portfolio was successfully deployed and accessible online.
This process helped me understand basic web deployment concepts and how GitHub
can be used as a professional hosting platform. I now feel confident deploying
and maintaining online portfolios and I know that employees can find me with the Portfolio.
